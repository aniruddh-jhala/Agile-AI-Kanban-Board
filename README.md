# 🚀 Enterprise Agile Kanban Board (Gen-AI Powered)

An interactive, production-ready Kanban Board designed for Product Managers and Agile teams. This application features drag-and-drop workflow states, persistent local storage, and real-time Gen-AI capabilities to optimize raw feature ideas into professional Jira ticket titles.

- **🔗 Live Project Link:** https://classy-bavarois-a105b9.netlify.app/
- **🛠️ Tech Stack:** HTML5, CSS3, Vanilla JavaScript (ES6+), Google Gemini 1.5 Flash API, LocalStorage API.

## ✨ Core Features
- **Dynamic Kanban Workflow:** Seamless drag-and-drop functionality between `TO-DO`, `IN PROGRESS`, and `DONE` states with dynamic task counters.
- **AI-Powered Product Management:** Integrated **Google Gemini 1.5 Flash API** to asynchronously map and convert raw user feedback/notes into enterprise-ready User Stories.
- **Fail-Proof Architecture:** Built-in client-side structural parsing engine acting as a zero-latency fallback algorithm during network dropouts.
- **Premium Performance UI:** Responsive glassmorphic dark theme built purely with semantic HTML5 and custom CSS3 keyframe animations.
